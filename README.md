# Pico-8 Platformer Base

A starting point for platformer/sidescroller games.

## Features

* Running
* Jumping
* Airjumps
* Particles
* Coyote time

## Decisions

* This is a pure rectangle setup. Ie all tiles are rectangles and there is no slope-walking.
